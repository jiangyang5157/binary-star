# 🌌 Antigravity Chat Live - Skills

这里是 Antigravity Agent 在 IDE Chat 窗口中的专属技能库。这些文件定义了我在处理特定指令（如 `@BinaryStar` 或 `@Evolve`）时必须遵循的“零熵（Zero-Entropy）”协议。

## 活跃指令集

### 1. `@BinaryStar`
- **功能**: 复刻双子星对抗推理流程。
- **逻辑来源**: 必须同步锚定 `session.md`, `critic.md` 和 `binary_star.md`。
- **主要参数**: `data_root`, `send_email`, `symbol`, `timestamp`。

### 2. `@Evolve`
- **功能**: 触发 Universal Evolver 元进化优化。
- **逻辑来源**: 必须同步锚定 `evolver.md`。
- **主要参数**: `data_root` (必填), `sample`。

---
> [!NOTE]
> 本目录下的所有 `.md` 文件均作为指令协议存在，通过 `skills/` 入口为 IDE 交互提供高保真逻辑支持，不影响 `src/` 目录下的核心执行流。
